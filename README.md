## 📌 Project Overview
This project analyzes sales and operational data from a US Superstore dataset (9,800 records). The main goal is to explore the data, understand patterns, and extract insights that can help improve business decisions.

---

## 🎯 Objectives
- Analyze sales trends over time  
- Identify top-performing products and categories  
- Explore customer behavior and shipping preferences  
- Understand regional sales distribution  
- Evaluate shipping efficiency  

---

## 🛠️ Tools & Technologies
- Python  
- Pandas  
- Matplotlib  
- Seaborn  

---

## 🔍 Data Analysis Process

### 1. Data Cleaning
- Handled missing values  
- Removed duplicates  
- Converted data types (especially date columns)  

### 2. Exploratory Data Analysis (EDA)
- Analyzed sales trends over time  
- Identified top products and categories  
- Explored customer segments and shipping methods  
- Compared performance across different regions  

### 3. Data Visualization
- Created charts to show trends and comparisons  
- Used Matplotlib and Seaborn for clear visual insights  

---

## 📊 Key Insights
- Sales increase clearly during November and December (seasonal trend)  
- Technology products generate the highest revenue  
- A small number of products contribute to a large portion of total sales  
- Most customers prefer Standard Class shipping, showing preference for lower cost over faster delivery  
- Sales performance varies across regions, with some regions contributing more to overall revenue  

---

## 🚀 Conclusion
Through this analysis, meaningful patterns in sales performance, customer preferences, and seasonal trends were identified. These findings highlight how data can be used to support decision-making and improve business strategies.
